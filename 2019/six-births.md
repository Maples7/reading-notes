# 浮生六记

Author: 沈复

沈复此书，书名虽然源自李白的《春夜宴桃李园序》，但比李白多三分静气。 “夫天地者，万物之逆旅；光阴者，百代之过客。而浮生若梦，为欢几何？” 李白行文有暗夜气，月光气，灯盏璀璨，又花枝招展，有一种秉烛夜游的急切。而沈复之文，有着落花流水的时光散漫，庭园梦境的从容静寂，与布衣蔬食的晨光之美。
